title: Tools
---

Here is a list of popular AI-powered coding tools that can help you code with an "AI vibe":

- [**GitHub Copilot**](https://github.com/features/copilot): An AI pair programmer that offers autocomplete-style suggestions as you code. It's great for quickly generating boilerplate code and learning new libraries.

- [**Windsurf (formerly Codeium)**](https://windsurf.com/): A powerful AI code editor that combines a deep understanding of your codebase with a suite of advanced tools to help you code faster and more efficiently.

- [**Tabnine**](https://www.tabnine.com/): A context-aware AI assistant that provides personalized code suggestions based on your coding patterns and best practices. It supports a wide range of languages and IDEs.

- [**Kodu.ai**](https://www.kodu.ai/): An AI coding companion that can autonomously build software from a description. It's a great tool for quickly prototyping ideas and getting a project off the ground.
