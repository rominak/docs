title: Install Cursor
---

Welcome to the Quickstart guide. Follow these steps to get started with Cursor.

### 1. Download Cursor

The first step is to download the Cursor application for your operating system.

- **macOS**: [Download for macOS](https://cursor.sh/)
- **Windows**: [Download for Windows](https://cursor.sh/)
- **Linux**: [Download for Linux](https://cursor.sh/)

Visit the official Cursor website and download the appropriate installer for your system.

### 2. Install Cursor

Once the download is complete, follow the installation instructions for your operating system.

#### macOS

1.  Open the downloaded `.dmg` file.
2.  Drag the `Cursor.app` file into your `Applications` folder.
3.  Launch Cursor from your `Applications` folder.

#### Windows

1.  Run the downloaded `.exe` installer.
2.  Follow the on-screen instructions to complete the installation.
3.  Launch Cursor from the Start Menu or desktop shortcut.

#### Linux

1.  The download will be an `AppImage` file.
2.  Make the `AppImage` file executable:
    ```bash
    chmod +x /path/to/your/Cursor.AppImage
    ```
3.  Run the `AppImage` to start Cursor:
    ```bash
    ./path/to/your/Cursor.AppImage
    ```

### 3. All Set!

You have successfully installed Cursor. You can now start exploring its features and coding with AI.
